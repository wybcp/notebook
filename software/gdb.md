# [GDB](https://www.sourceware.org/gdb/documentation/)

GDB: The GNU Project Debugger

GDB 是一个由 GNU 开源组织发布的、UNIX/LINUX 操作系统下的、基于命令行的、功能强大的程序调试工具。

```bash
$ brew install gdb
$ gdb --version
```
